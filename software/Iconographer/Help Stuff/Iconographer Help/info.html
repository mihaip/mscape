Iconographer 1.0
Created by: Mihai Parparita
Published by: Mscape Software

Who is to blame?

    Arlo Rose provided the impetus to start this program. Eventually, under the development name of Icon Mangler, it became my Computer Science IB Higher Dossier project, and was submitted to the commission on April 14 (scores are still pending). After a month-long break, I started programming again, and added four more tools. Then Arlo showed his head again. Based on certain suggestions made by him, Iconographer acquired Windows and Mac OS X Server icon support, a new icon and about box, pattern support, and other little tweaks.

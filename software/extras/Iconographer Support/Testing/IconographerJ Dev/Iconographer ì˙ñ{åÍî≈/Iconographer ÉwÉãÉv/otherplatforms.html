Windows アイコン

    “Iconographer”は、Windows のアイコン形式である【.ico】ファイルの読み書きをサポートしています。“Iconographer”で編集保存されたアイコンファイルは、Windows 95/98 上で使用可能です。アイコンファイルの読み込みはメニューバーのファイル /開く/ Windows アイコン からファイルを選択することによって、他のアイコンと同様にアイコン編集ウインドウにアイコン画像が表示されます。注意しなければならないのは、Windows アイコン形式は 8 bit マスクをサポートしていないので、 8 bit マスクのデータは保存時に失われてしまいます。
MacOS X Server アイコン

    “Iconographer”は、QuickTime 4.0 がインストールされている場合のみ、MacOS X Server のアイコン形式である【.tiff】ファイルの読み込みをサポートしています。アイコンファイルの読み込みはメニューバーのファイル /開く/ Mac OS X アイコン からファイルを選択することによって、他のアイコンと同様にアイコン編集ウインドウにアイコン画像が表示されます。現在のバージョンでは、この形式のアイコンを保存することは出来ませんが、将来のバージョンでは保存出力可能になるでしょう。

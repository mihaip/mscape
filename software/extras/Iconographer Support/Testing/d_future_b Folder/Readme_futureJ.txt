 [ FUTURE ICONS Ver0.2 - 99/07/29 ]

-----------
 D_Future.icl はIconLiblalyFile です。
 拡張子を *.iclから*.dllに変更すれば、DLLファイルとしても使用できます。

 D_Future.ips は KH IconMagic98 で使用する設定ファイルです。
 この設定ファイルを使えば、アイコンの変更が簡単に行えます。
 (自動的に変更されないアイコンは手動で変更して下さい。)

 (注) D_Future.icl を C:\WINDOWS\SYSTEM フォルダに入れてから使用して下さい。
       その際にファイル名を変更しないで下さい。
       ドライブレターやフォルダ名が違う場合は、各自の環境に合わせて直接
       メモ帳等で書き換える必要があります。

[ KH IconMagic98 ]  http://www.vector.co.jp/authors/VA007769/

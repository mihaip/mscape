Icon Machine Tools

    Icon Machine has a pretty much standard array of painting tools: lasso, marquee, pencil, paint bucket, line, rectangle, oval, line, polygon, and text. The usual rules for each tool apply- use the shift key to constrain them, etc.

    When using the text tool, the arrow keys don't move the insertion point; instead, they move all of the text by one pixel.

    You can also use the keyboard to select the drawing tools. These are mostly the same shortcuts used by Adobe Photoshop.

    L - Lasso M - Marquee
    K - Bucket Y - Pencil
    T - Text E - Eraser
    O - Oval R - Rectangle
    N - Line P - Polygon
    I - Eyedropper

    It may not be obvious, but there is a "magic wand" selection tool - just select the lasso and hold down the option key. In a future version the magic wand will get its own spot on the tool bar; there isn't room right now.

    Various special things have been added to the tools for working with masks. For more information, see the Icon Masks section.

Editing Masks

    Icon Machine edits the icon and the mask simultaneously, and also has a Mask View for focusing on just the mask. To switch between Normal View and Mask View, use the buttons in the top right of the window, or press the Tab key.

    The little dots you see in the background in Normal View are indicators of the mask for each pixel. The more visible the dot is, the more transparent the pixel. To hide the dots, choose the Show Mask command from the View menu to uncheck it.

Drawing Modes

    In Normal View, all of the drawing tools affect both the icon and the mask by default. You can change this by choosing from the different drawing modes. Normal drawing mode is the default behavior of editing both the icon and the mask. Mask mode affects only the mask; Image-Only mode affects only the icon image. To quickly switch to Mask mode, hold down the Command key. To quickly switch to Image-Only mode, hold down the Command and Option keys.

    Not all tools are affected by the current drawing mode. Those that do will display a different cursor for each mode: solid black for Mask mode, and hollow for Image-Only mode.

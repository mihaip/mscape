 [ CYBER ICONS ]

-----------
 D_Cyber.icl はIconLiblalyFile です。
 拡張子を *.dllに変更すれば DLLファイルとして使用できます。
 
 D_Cyber.ips は KH IconMagic98 で使用する設定ファイルです。
 これを使えば変更が簡単に行えます。

 (注) D_Cyber.icl を C:\WINDOWS\SYSTEM フォルダに入れてから使用して下さい。
       その際にファイル名を変更しないで下さい。
       ドライブレターやフォルダ名が違う場合は、各自の環境に合わせて直接
       メモ帳等で書き換える必要があります。

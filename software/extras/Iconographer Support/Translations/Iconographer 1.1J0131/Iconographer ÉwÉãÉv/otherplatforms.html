Windows 形式アイコン

    “Iconographer”は、Windows のアイコン形式である【.ico】ファイルの読み書きをサポートしています。“Iconographer”で編集保存されたアイコンファイルは、Windows 9x/NT 上で他のアイコンと同様に使用可能です。Windows のアイコンファイルの読み込みはメニューバーのファイル /開く/ から表示されるダイアログのファイル形式のポップアップメニューで Windows 形式 を選択してからファイルを選択することによって、他のアイコンと同様にアイコン編集ウインドウにアイコン画像が表示されます。注意しなければならないのは、Windows アイコン形式は 8 bit マスクをサポートしていないので、 8 bit マスクのデータは保存時に失われてしまいます。

MacOS X サーバー形式アイコン

    “Iconographer”は、QuickTime 4.0 以降がインストールされている場合のみ、MacOS X サーバー のアイコン形式である【.tiff】ファイルの読み込みをサポートしています。アイコンファイルの読み込みはメニューバーのファイル /開く/ から表示されるダイアログのファイル形式のポップアップメニューで Mac OS X サーバー形式 を選択してからファイルを選択することによって、他のアイコンと同様にアイコン編集ウインドウにアイコン画像が表示されます。現在のバージョンでは、この形式のアイコンを保存することは出来ませんが、将来のバージョンでは保存出力可能になるでしょう。

Iconographer 1.1 日本語版
作者：Mihai Parparita
配付：Mscape Software

誰に責任があるのか？

    Arlo Rose はこのプログラムを始めるにあたってそのきっかけを与えてくれました。“Icon Mangler”という開発名で4月14日に私のComputer Science IB Higher Dossier プロジェクトに委任、提出されました。1ヶ月の空白期間の後、私はプログラミングを開始し4つのツールを追加しました。そして Arlo に再びそれを見せました。元々の提案者は彼ですが、“Iconographer”は、 Windows と MacOS X Server のアイコンをサポートし、新しいアイコンと情報ダイアログ、パターンのサポート、その他いくつかの追加点がありました。“Iconographer”の開発途中での Arlo の提案や意見に大変感謝します。

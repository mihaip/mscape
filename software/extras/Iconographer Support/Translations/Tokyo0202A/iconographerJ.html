　-Iconographer についてー
“Iconographer”は、デスクトップに表示される各種アイコンや、各種プログラムで使用されているリソース内のアイコンを変更することが出来るアイコン編集ソフトウェアです。“Iconographer”はMacOS 8.5で追加された32-bit のフルカラーアイコンと8-bit のグレースケールアイコンマスクを何もない状態から新規に作成することが可能です。また追加点としてMacOS以外のプラットフォームのアイコンも編集することが出来ます。(現在、編集と入出力可能なアイコンには、“Windows”の【.ico】ファイルと“Mac OS X サーバー”の【.tiff】ファイルがあります。) アピアランスマネージャがインストールされている Power Macintosh の各機種で動作します。

“Iconographer”は、無料のフリーウェアではなく登録使用料の必要なシェアウェアです。もしより多くのアイコンを編集したいと思うならば $15 の登録料 の登録料金を支払う必要があります。

Iconographer は、他に 英語版 (English) と 仏語版 (en français) があります。

　-機能説明ー
- Mac OS 8.5 以降で採用された 32-bit アイコンの完全サポート
- Windows の【.ico】アイコンファイルの読み込み書き込み
- Mac OS X Server の【.tiff】アイコンファイルの読み込み(48 x 48 サイズのアイコンを含む)
- 描画の無段階の取り消し操作 (限界はメモリの許す限り)
- 強力な選択ツールと操作性
- 多彩なカラーパレット
- アピアランスマネージャー完全対応
- 複数のアイコンリソースを含むファイルの容易な編集
- 無段階の拡大表示
- わかりやすい色変換s
- アンチエイリアシングのサポート
- Photoshop や他のグラフィックプログラムとの互換性確保
- 使用するのに Mac OS 8.5 は必要ありません
- 詳しい説明書類とバルーンヘルプの採用
- 英語版 、日本語版 、仏語版 の各国語版がリリースされています

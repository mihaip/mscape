Les outils de dessin

    Les outils de dessin opèrent normalement sur toute l'icone, sauf dans le cas où une partie de l'icone est sélectionnée. Dans ce cas, les outils de dessin sont limités aux frontières de la sélection.

Crayon

Raccourci-clavier : N
Compléments : La touche Option transforme momentanément le crayon en pipette. La touche majuscule force le déplacement du crayon à l'horizontale ou la verticale.
Description: L'outil crayon agit quand la souris est appuyée et déplacée sur la surface de dessin. Il colorie les pixels le long du trajet de la souris. En temps normal, il trace le trajet avec la couleur de premier plan, mais si le premier pixel du trajet était déjà de cette couleur, alors c'est la couleur d'arrière-plan qui est utilisée.

Pipette

    Raccourci-clavier : I
    Compléments : Appuyez sur la touche Option pour récupérer la couleur comme couleur d'arrière-plan.
    Description: L'outil Pipette est utilisé pour récupérer une couleur déjà présente dans le dessin. Placez le curseur sur un pixel et cliquez afin que la couleur de ce pixel devienne la couleur de premier plan.

Pot de peinture

    Raccourci-clavier : K
    Compléments : La touche Option transforme momentanément l'outil en pipette.
    Description : Le pot de peinture est utilisé pour remplir des surfaces avec la couleur de premier plan et d'arrière plan (le mélange des deux est déterminé par le sélecteur de motif). La surface à remplir est déterminée par la surface de points contigus de même couleur que le pixel cliqué.

Gomme

    Raccourci-clavier : E
    Double-Clic : efface toute l'icone
    Compléments : La touche Option transforme momentanément la gomme en pipette. La touche Majuscule force le déplacement de la gomme à l'horizontale ou la verticale.
    Description : La gomme fonctionne exactement comme le crayon, sauf qu'elle utilise la couleur d'arrière plan pour remplir les pixels.

Rectangle

    Raccourci-clavier : R
    Double-Clic : Passage du mode rectangle vide au mode rectangle plein (rempli par la couleur d'arrière-plan et le motif choisi dans le sélecteur de motifs).
    Compléments : Avant de cliquer, la touche Option transforme momentanément le rectangle en pipette. Après avoir cliqué, la touche option permet de dessiner le rectangle centré autour du point initial. La touche Majuscule permet de dessiner des carrés à la place de rectangles.
    Description : L'outil Rectangle dessine un rectangle dont l'un des coins est le point initial (là où vous avez cliqué) et dont le coin opposé est le coin où vous relâchez la souris.

Ovale

    Raccourci-clavier : O
    Double-Clic : Passage du mode ovale vide au mode ovale plein (rempli par la couleur d'arrière-plan et le motif choisi dans le sélecteur de motifs).
    Compléments : Avant de cliquer, la touche Option transforme momentanément l'ovale en pipette. Après avoir cliqué, la touche option permet de dessiner l'ovale centré autour du point initial. La touche Majuscule permet de dessiner des ronds à la place d'ovales.
    Description: L'outil Ovale dessine un ovale dont l'un des coins est le point initial (là où vous avez cliqué) et dont le coin opposé est le coin où vous relâchez la souris. Cet outil supporte l'anti-aliasing (lissage des bords du dessin) quand cette option est activée.

Traits

    Raccourci-clavier : Y
    Compléments : La touche Option transforme momentanément l'outil en pipette. La touche majuscule force le trait à suivre des intervalles de 45°.
    Description: L'outil Traits dessine un trait reliant le point initial (là où vous avez cliqué) au point final (là où vous avez relâché la souris). Cet outil supporte l'anti-aliasing (lissage des bords du dessin) quand cette option est activée.

Polygone

    Raccourci-clavier : P
    Double-Clic : Passage du mode polygone vide au mode polygone plein (rempli par la couleur d'arrière plan et le motif choisi dans le sélecteur de motifs).
    Compléments : La touche Option transforme momentanément l'outil en pipette. Après le clic, la touche majuscule force le trait dessiné à suivre des intervalles de 45°.
    Description : L'outil Polygone est utilisé pour dessiner des polygones aux formes irrégulières. Le premier clic désigne le point initial, chaque clic ajoute un sommet au polygone. Pour compléter le polygone, cliquez a nouveau sur le point initial ou double-cliquez afin de relier le point actuel et le premier sommet. Cet outil supporte l'anti-aliasing (lissage des bords du dessin) quand cette option est activée.

Dégradé

    Raccourci-clavier : G
    Double-Clic : Passage du mode dégradé linéaire au mode dégradé concentrique.
    Compléments : La touche Option transforme momentanément l'outil en pipette. Après avoir cliqué, la touche Majuscule force le dégradé à suivre des intervalles de 45°.
    Description : L'outil Dégradé permet de dessiner des dégradés entre la couleur d'arrière-plan et la couleur de premier plan. De la même manière que pour l'outil Traits, le dégradé est dessiné entre un point initial et un point final.

Texte

    Raccourci-clavier : T
    Compléments : La touche Option transforme momentanément l'outil en pipette.
    Description: L'outil Texte permet de dessiner du texte sur l'icone. Après le clic, un dialogue vous permet de choisir la police, la taille, le style et le texte. Le texte sera inséré sur l'icone là où vous avez cliqué dès que vous aurez refermé cette fenêtre de dialogue.

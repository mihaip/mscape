Icones Windows

    Iconographer supporte l'ouverture et l'enregistrement des icones au format .ico de Windows. Le fichier ainsi créé peut être transféré sur une machine sous Windows 9x/NT/2000, où il pourra être utilisé comme n'importe quel fichier d'icone. Pour ouvrir une icone Windows, choisissez l'option Windows dans le menu Format, dans la fenêtre d'ouverture de fichier (menu Fichier > Ouvrir). Une fois le fichier choisi, il sera placé dans l'éditeur d'icone, et vous pourrez le travailler comme une icone quelconque. Notez que le format Windows ne supporte pas le masque 8-bits, et que celui-ci serait perdu à l'enregistrement. Ces icones ont les mêmes possibilités et limites que celles du format Mac OS Ancien.

Icones Windows XP

    Windows XP utilise un format d'icones très proche de son prédecesseur (fichier .ico), mais avec quelques améliorations. Les masques 8-bit sot maintenant supportés, pour permettre des effets de transparence, d'ombre et autres. De plus, le format 48x48 prend de l'importante, car il est plus souvent utilisé dans le système. Cela en fait un format très proche des formats MacOS X Server 1.x et Mac OS Nouveau.

Icones Mac OS X Server 1.x

    Iconographer supporte l'ouverture des icones .tiff de MacOS X Server 1.x, quand QuickTime 4 est installé. Pour ouvrir une icone MacOS X Server, choisissez l'option MacOS X Server dans le menu Format dans la fenêtre d'ouverture de fichier (menu Fichier > Ouvrir). Une fois le fichier choisi, il sera placé dans l'éditeur d'icones, et vous pourez le travailler comme une icone quelconque. Notez que ce format est celui de MacOS X Server lancé en 1999, qui n'est pas le même que l'actuel MacOS X Server. Pour créer des icones pour MacOS X Server version 10.0 et supérieure, enregistrez vos icones dans le format Mac OS X.

Les outils de navigation

    Ces outils permettent de se déplacer dans l'icone et de changer son affichage.

Main

    Raccourci-clavier : H
    Double-clic : Zoome sur l'icone de sorte à l'afficher en entier, sur la plus grande surface possible.
    Description : l'outil Main permet de se déplacer dans l'icone affichée. En cliquant puis en déplaçant la souris, la partie visible est déplacée dans la direction de la souris (cela correspond également à un déplacement de la fenêtre de travail sur l'icone).

Loupe

    Raccourci-clavier : Z
    Double-clic : Passe l'icone en affichage à 100% (taille réelle)
    Compléments : La touche Option rétrécit au lieu d'aggrandir. La barre d'espace permet de passer temporairement à l'outil Main.
    Description : L'outil Loupe permet de modifier l'aggrandissement de l'icone. Cliquer permet de doubler le niveau de zoom en centrant la fenêter sur le point désigné.

A quoi sert Iconographer ?

    Iconographer est un éditeur d'icone. Il vous permet de changer les icones des fichiers mais aussi d'accéder aux icones contenues dans les ressources de nombreux programmes. Iconographer a été prévu dès le début pour gérer les nouvelles icones de MacOS 8.5 avec leurs millions de couleurs et leurs masques 8-bits. En plus, vous pouvez travailler avec des icones d'autres plates-formes (.ico de Windows et .tiff de MacOS X Server). Iconographer fonctionne sur n'importe quel Power Macintosh équipé de l'Appearance Manager.

    Iconographer est un shareware : si vous l'utilisez régulièrement, merci de payer les 15$ d'enregistrement. Pour plus de détails, reportez-vous à la page d'informations sur l'enregistrement.

Comment utiliser Iconographer ?

    Iconographer respecte les directives d'Apple concernant l'interface graphique. Si vous avez déjà utilisé un quelconque logiciel de dessin, vous ne serez pas dépaysé par Iconographer. Une session de travail ressemblera à cela :

    1

    Ouvrir un fichier ou un dossier avec Iconographer.

    Vous pouvez glisser un fichier sur l'icone d'Iconographer ou utiliser la commande "Ouvrir" du menu Fichier.

    2

    Editer l'icone avec les outils.

    Plus d'informations dans la section "Outils".

    3

    Enregistrer l'icone et quitter Iconographer.

    Iconographer enregistre les icones dans les deux formats (MacOS 8.5 et ancien format), afin que tous les utilisateurs puissent les visualiser. Iconographer sait aussi enregistrer les icones dans le format Windows (.ico).

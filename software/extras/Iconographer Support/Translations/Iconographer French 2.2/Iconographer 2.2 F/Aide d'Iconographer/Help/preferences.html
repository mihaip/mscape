Préférences

    Les préférences (dans le menu Edition) permettent de contrôler certaines fonctions d'Iconographer. Elles sont divisées en 3 catégories: Général, Defaut, et Editeur externe. Voici leurs descriptions.

Général

    N'afficher que les membres chargés : Par défaut, la palette des membres affichera tous les membres supportés par le format de l'icone ouverte. En activant cette option, vous limitez la palette aux membres contenus dans l'icone.

    Vérifiez les masques à l'enregistrement : Si les masques sont créés de manière incorrecte, l'icone peut ne pas s'afficher correctement dans le Finder. Cette option permet à iconographer de vérifier les masques et, au besoin, de corriger les erreurs.

    Pixelliser les couleurs : cette option pixellise les images dans le cas où elles sont utilisées avec des profondeurs de couleurs inférieures à l'origine (par exemple en collant une image 32-bits dans la zone 8-bits). Pixelliser, c'est utiliser un motif pour simuler des couleurs.

    Options du démarrage : cette série de boutons vous permet de choisir quelle action Iconographer doit faire à son démarrage. Par défaut, il crée une nouvelle icone vierge, mais il peut aussi ouvrir une icone ou simplement rester ouvert et ne rien faire.

    Affichage de la palette des membres : La palette des membres affiche ceux-ci à leur taille réelle. Cependant, si les membres sont nombreux et la place limitée, naviguer dans la palette peut devenir ennuyeux. Cette option permet de réduire la taille des membres (il s'agit d'une opération visuelle, le contenu de l'icone n'est en rien modifié).

Défaut

    Zoom par défaut : ce réglage détermine le niveau de zoom utilisé à l'ouverture d'un icone. Vous pouvez changer le niveau de zoom pour chaque éditeur, mais si vous avez l'habitude de zoomer sur toutes les icones que vous ouvrez, fixez ce réglage une fois pour toutes.

    Format d'icones par défaut : Cette option détermine le format des nouvelles icones créées, ainsi que le choix par défaut dans les fenêtres d'ouverture et d'enregistrement.

    Membres par défaut : Même si les membres d'une icone peuvent être gérés au cas par cas grâce à la fenêtre d'infos, cette option vous permet de les régler pour chaque nouvelle icone.

Editeur externe

    Editeur externe: Cliquez sur ce bouton pour choisir l'application qui servira d'éditeur externe.

    Raccourci Editeur externe : Pour faciliter le passage entre Iconographer et l'éditeur externe, un raccourci-clavier est autorisé. Pour le choisir, cliquez sur ce bouton tout en appuyant sur toutes les touches que vous souhaitez utiliser.

    Format d'export : Pour transférer les données de l'icone vers l'éditeur, Iconographer doit créer un fichier temporaire au format sélectionné. Toutes les applications ne gèrent pas tous les formats.

    Exporter l'icone et le masque : Si vous souhaitez éditer à la fois l'icone et le masque, cochez cette option. Le masque sera affiché en tant que couche alpha dans le document.

    Pour plus d'informations sur l'utilisation des éditeurs externes, allez voir la section sur le travail avec d'autres programmes.

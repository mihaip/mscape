[プレスリリース] Iconographer 1.1日本語版：32-bit アイコン編集ソフトウェア“Iconographer”は、32-bit のフルカラーアイコンに完全対応したアイコン編集ソフトウェアです。バージョン1.1にアップデートされて以下のような新機能が追加されました。- フローティングカラーパレットを追加- 複数のアイコンリソースを含むファイルのためのアイコンブラウザの改善- アンチエイリアシングのサポート- ウインドウメニューの追加- Windows 形式アイコンのサポート改善- 3 つに分かれていた、開くコマンドを1つにしました。- 多くのバグ修正と使いやすさを改善しました。- 現在、3種類の言語に対応しています：英語、日本語、仏語Iconographer の製品情報は：http://www.mscape.com/products/iconographerJ.htmlIconographer のダウンロードは：http://www.mscape.com/products/downloads/iconographerJ.sit.hqxIconographer について“Iconographer”は登録料金たったの $15 のシェアウェアアイコン編集ソフトウェアです。いろいろな他のアイコンリソース編集プログラムと同様に、アイコンを変更する事が出来ます。“Iconographer”はMacOS 8.5で追加された32-bit のフルカラーアイコンと8-bit のグレースケールアイコンマスクを何もない状態から新規に作成することが可能です。また追加点としてMacOS以外のプラットフォームのアイコンも編集することが出来ます。(現在、編集可能なアイコンには、“Windows”の【.ico】ファイルと“Mac OS X Server”の【.tiff】ファイルがあります。)複数回の取り消し、色調整ツール、アイコンブラウザ、バルーンヘルプ、HTML形式のヘルプシステム、これらによって快適なアイコン編集をすることができるでしょう。システム必要条件- PowerPC を搭載した Power Macintosh 各機種- 漢字Talk 7.5.3 以降のシステムソフトウェア- アピアランスマネージャ (Mac OS 8.0 以降ではシステムに組み込まれています)Mscape Software についてMscape Software は、MacOSコミュニティーに不足している思われるツールの開発のために1998年に創立されました。最初の製品“clip2cicn”は、Kaleidoscope 2.0 形式のスキームを作成するために必要不可欠なツールとしてスキーム作者達に愛用されてきました。“clip2icns”は、1998年10月に MacOS8.5で追加された32-bit のフルカラーアイコンを作成する(実際には、他のグラフック作成プログラムから画像部分は作成しなくてはいけませんでしたが)最初のツールとして登場しました。その後すぐに、姉妹製品“Glypher”は、簡単にフォルダアイコンを作成するためにリリースされました。Mscape Software は、現在 MacOS X やOpenGL のようなテクノロジーを含む先進的な製品の開発に取り組んでいます。

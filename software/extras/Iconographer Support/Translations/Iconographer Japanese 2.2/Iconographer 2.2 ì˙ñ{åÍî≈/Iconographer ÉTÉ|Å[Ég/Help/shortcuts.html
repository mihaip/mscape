より詳しいヘルプ情報の入手

    Mac OS 8.x / Mac OS 9.x 使用時にはこの HTML 形式ヘルプに加えて“Iconographer”は、バルーンヘルプもサポートしています。ヘルプメニューからバルーン表示を選択することによって、機能がよくわからない部分（“Iconographer”で表示されるすべてのメニュー、ダイアログ、項目など）にマウスカーソルを合わせると、バルーンポップアップにより詳しい機能の説明が表示されます。

    もしそれでもまだ、“Iconographer”でわからない部分がある場合には、作者：iconographer@mscape.comまで英語にて連絡して下さい。あなたを助けるために最大限努力します。

キーボードショートカット

    すべての選択ツールや描画ツールはキーボードショートカットを持っています。これについての詳しい説明は、ツールの使い方 ページを御覧下さい。

    選択範囲の削除の時に通常は、削除した部分は背景色で塗りつぶされますが別のショートカットとして削除時（またはフロート状態を削除時）に【オプション】キーを押すことによって削除した部分を前景色で塗りつぶすことが出来ます。

    【tab】キーを押すことによって現在編集中のアイコン画像とそのマスク画像とを簡単に切り替えることが出来ます。

    (日本語翻訳者注：日本語版 Mac OS を使用している場合、キーボードショートカットは英字入力モード時のみ有効ですので、メニューバー右の入力メニューを“US”に切り替えて下さい。）

ドラッグアンドドロップ

    “Iconographer”は、 その他のプログラムからのドラッグアンドドロップをサポートしています。あるアイコンを拡大縮小するために他へコピーする場合ドラッグアンドドロップでコピーが可能です。マスク画像を作成する際に、完成したアイコン画像をマスク画像にドラッグアンドドロップするとアイコン外側の白色部分を除外したおおまかな形状のマスクがコピーされるので、あとは簡単な修正のみで使用できます。

その他

    Finder上でのアイコン選択状態をプレビューしたい場合、単純にプレビュー枠のアイコンをクリックするだけで Finder上と全く同じ状態にプレビューされます。選択状態を解除するにはもう一度アイコンをクリックして下さい。

Iconographer とは?

    “Iconographer”は、デスクトップに表示される各種アイコンや、各種プログラムで使用されているリソース内のアイコンを変更することが出来るアイコン編集ソフトウェアです。“Iconographer”は数多くのフォーマットのアイコンを作成することができます。classic Mac OS (8 ビットアイコンのみ)、Mac OS 8.5+ (8 ビットマスクを含む 32 ビットアイコン)、Mac OS X (128 x 128 サイズのアイコン .icns ファイル)、Windows, Windows XP(.ico ファイル)、Mac OS X Server (48 x 48 サイズのアイコン .tiff ファイル)アピアランスマネージャがインストールされている Power Macintosh の各機種で動作します。(“Iconographer”は Mac OS X 環境下では、Classic アプリケーションとして動作します。)

    “Iconographer”は、無料のフリーウェアではなく登録使用料の必要なシェアウェアです。もしより多くのアイコンを編集したいと思うならば $15 の登録料金を支払う必要があります。 “Iconographer”のシェアウェア登録は、Kee Nethery氏 によって運営されているオンライン登録処理代行会社 Kagi によって処理されます。登録の仕方を御覧下さい。

Iconographer はどのように使用するのですか?

    “Iconographer”は、“MacOS ヒューマンインターフェースガイドライン”に沿って作成されていますので、あなたが以前にどうのようなグラフィック編集ソフトウェアを使用していた場合でも同様の操作で使用出来るようになるでしょう。編集の基本的な説明は以下のようになります：

    1 ファイルやフォルダを“Iconographer”で開く
    編集したいファイルやフォルダを“Iconographer”アイコンにドラッグアンドドロップするか、ファイルメニューの“開く...”からアイコンを選択して下さい。
    2 提供されたツールでのアイコンの編集
    より詳しい説明は ツールの使い方 を御覧下さい。
    3 アイコンを保存して“Iconographer”を終了する
    “Iconographer”は MacOS 8.5で追加された32-bit のフルカラーアイコンとそれ以前のシステムでのアイコンの両方をサポートしています。これによってどのようなシステムを使用している人にもあなたの作成したアイコンを見てもらうことが出来ます。 また追加点としてMacOS以外のプラットフォームのアイコン“Windows”の【.ico】ファイルなども編集することが出来ます。

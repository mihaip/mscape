開発秘話

    時は Mac OS 8.5 リリース直後、Arlo Rose が新フォーマットのアイコンを編集するツールを作ってくれないかとの依頼がありました。その結果 clip2icns や Glypher を作成しました。しかし、彼らにとってこのツールは、Mac のユーザーインターフェース製作の補助程度のものでしかありませんでした。同時期、私は IB コンピューター科学科に属していました。たくさんの技術を披露するためのポートフォリオプログラム作成をしていました。予想可能なデータベースプログラムを作成するかわりに、学術的な感覚より多くの役に立つ何かをすることができたと気がつきました。そのようにして、“Icon Mangler”は誕生しました。数カ月後 (特に、私の冬休みの間) 私にとって最初の Macintosh アプリケーションが完成しました。４月の終わりに、私は１４０ページに及ぶソースコード、４０ページの仕様書、設計書、テストデータのすべてをプリントアウトし、成果を楽しみにしているであろうIB 評議委員会に提出しました。("これは何だ？アイコン編集ソフトウェア？Mac 用の？ハハハッ！何人かのクリエーターにとってはいいものだろうね。")その結果、私は最高の成績を得ることになりました。

    夏休みが始まった頃、私は“Iconographer”のソースコードをチェックしましたが、それはまだリリースの段階ではないと気がつきました。Arloは、グラフィック作業に戻り、彼のおかげで“Iconographer”は、Windows アイコンのサポートと合理的で洗練されたユーザーインターフェースを手に入れました。夏休みが終わった頃に、私は、“Iconographer”のリリースは近いであろうと判断しました。なぜならばこれ以上この仕事をしている時間がなかったからです。このリリース版では基本的な要素のみのものでした。(大幅なアップデートは次の年の２月までありませんでした)しかし早急にリリースする必要があったため仕方がありませんでした。バージョン1.0.1 は、その翌日にリリースしましたがそれでさえすべてのバグを解決することはできず、バージョン1.0.2 は、その２日後に続けてリリースしました。大学が重要な時期になったため、それからしばらくの間“Iconographer”はアップデートをしませんでした。少し時間が取れるようになり、バージョン1.0 に入れたかったいくつかの仕様(アイコンブラウザとフローティングカラーパレットの改善、アンチエイリアスやインターフェースの改善など)を追加しようと思いましたが、それには十分な時間はありませんでした。バージョン1.1 は、最初の他言語リリースになりました。Yoshi SAKUMA、Baptiste Leheup の協力に感謝します。彼らは日本語版、フランス語版の翻訳に関しての権利を所有しています。

    最近になり、Mac OS X が波にのってきました。それにより１つの仕様変更がありました。新しいアイコンはとても大きく128 x128 ピクセルのサイズになるということでした。当初、Apple 社にはこのアイコンを編集する正式なツールがなく“Iconographer”がその穴を埋める予定でしたが、私にはそのための時間がなくこの契約は実際には結ばれませんでした。しかしながら、私はここから Apple 社は、私の編集ツールと競合するソフトウェア (はーい David ！) を手に入れるところであったと知りました。黙って立っているわけにはいきませんでした。“Iconographer”を MacOS X の最初のアイコン編集ソフトウェアにするべく決心しました。可能性を検討した結果、新しいアイコンに対応するには、ユーザーインターフェースすべての作り直しをする以外に選択肢はありませんでした。このようにして現在の４つのパレットという仕様が生まれました。(アイコン作成者が余計な時間をさかなくてもすむように Adobe Photoshop を参考にしてデザインしました。) さらにアイコンの元になる画像を作成するために他の画像編集ソフトウェアとの親和を高めるために外部エディタのサポートを追加しました。“Iconographer”が編集するアイコンのあらゆる方法を確実にするために多くの微調整をしました。このリリースの主たる仕様変更が128 x128 ピクセルサイズのアイコンのサポートであったため、それ自信がユーザーインターフェースの変更改善を必要としていたのは明らかでした。このデザインための icons.cx の Rick Roe の協力に感謝します。

    最後に、Mac OS X のパブリックベータ版の発表があり自分自信にかせられた最終期限が迫っていました。そのためバージョン 2.0 のリリースを考え始めるようになりました。“Iconographer”を Classic アプリケーション(Mac OS 8.x / 9.x 用)としてリリースするということが最善だと私は思いました。しかし、カーボン化したバージョンをできる限り早くリリースしなければいけないと考えました。なぜならばApple は、新しい Mac への Mac OS X 標準インストールを開始したからです。

    Iconograoher X は今準備ができました。

バージョン履歴　　　

    バージョン2.2 2001年10月22日

    • Windows XP のアイコンファイルの読み込みと編集をサポートしました。
    • (.rsrc) を含むアイコンの編集をサポートしました。
    • マスクとアイコンの生成コマンドを追加しました。
    • 選択範囲の拡大／縮小コマンドを追加しました。
    • Mac OS X 環境下での速度改善とインターフェースの改善を行ないました。
    • その他多くの問題を修正しました。

    バージョン2.1 2001年09月05日

    • Iconographer は カーボン化され Mac OS X 上でネイティブソフトウェアとして動作します。
    • Mac OS X アイコン形式 (.icns) のファイル互換性を改善しました。
    • アイコン形式の変換を改善しました。
    • カラーパレットを再構築しました。
    • ライブウインドウのドラッグをサポートしました。 (Mac OS X 環境下でのみ)
    • その他多くの問題の修正とインターフェースの改良を行いました。

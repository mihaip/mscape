The Navigation Tools
    This set of tools is used to move around and change the way in which the current icon is displayed.
Pan
    Keyboard Shortcut: H
    Double-click shortcut: Sets the magnification such that it's at the maximum level that allows the entire icon to be visible.
    Description: The pan tool is used to scroll around the current icon. By clicking and dragging onto the canvas, the currently visible area is shifted in the direction of the drag (this can be thought of as holding onto the canvas and moving it).
Zoom
    Keyboard Shortcut: Z
    Double-click shortcut: Sets the magnification level to 100% (actual size).
    Modifiers: Option zooms in instead of out. The space bar temporarily puts the tool into panning mode.
    Description: The zoom tool is used to increase the magnification that the icon is viewed at. Clicking sets that area as the center of the of drawing and doubles the magnification (up to 1600%).

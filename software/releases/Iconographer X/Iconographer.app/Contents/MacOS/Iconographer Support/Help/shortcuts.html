Getting More Help

    In addition to this HTML-based help, when running under Mac OS 8.x/9.x, Iconographer also supports balloon help. Simply select Show Balloons from the Help menu, and then point at the thing that interests you (menus, dialogs, and items in the editing window all work), and more details will be given in the balloon that pops up.

    If you still have more questions about Iconographer, you can contact us at iconographer@mscape.com and we'll do our best to help you.
Keyboard Shortcuts

    All of the selection and drawing tools have shortcuts, they are mentioned on the tools page.

    Although it is probably obvious that delete fills a selection with the background color (or removes it if it is floated), another shortcut is to hold down option while deleting, to fill it with the foreground color.

    To quickly cycle between the current icon and mask, the tab can be used.
Drag and Drop
    Iconographer has extensive support for drag-and-drop, both within the program and from other programs as well. To scale one member to another, all one needs to do is to drag among them. In addition, to make a mask out of the icon contents (a sillouete will be generated), a simple drag from an icon slot into a mask slot is all that is necessary.
Others

    To display the preview in the selected state, simply click in the preview area. The icons there will be darkened in exactly the same manner that the Finder uses. Click again to remove the darkening.
